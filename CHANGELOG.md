## [0.1.0] - 2025-06-01

### Added

- Initial release of MCP NanoBanana Server
- Image generation tools:
  - `nanobanana_generate_image` - Generate image from text prompts
  - `nanobanana_edit_image` - Edit existing images
- Task tracking:
  - `nanobanana_get_task` - Query single task status
- Information tools:
  - `nanobanana_list_models` - List available models
- stdio and HTTP transport modes
- Comprehensive test suite
- Full documentation
